# HarNova — Enterprise Site

Marketing site for HarNova: services, product roadmap, and contact info.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed in the terminal.

## Build for production

```bash
npm run build
npm run preview
```

## Project structure

```
src/
  App.jsx        page content and layout
  index.css      global styles and CSS variables
  main.jsx       app entry point
public/
  favicon.svg    site icon
```
